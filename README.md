## Installation
### Cocoapods
```
pod 'GoogleMLKit/TextRecognition'
pod 'GoogleMLKit/TextRecognitionKorean', '2.3.0'
pod 'TesseractOCRiOS'
pod 'Alamofire'
pod 'AlamofireImage'
```
